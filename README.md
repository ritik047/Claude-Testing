# AI-Powered Merchant Onboarding for Payment Aggregators

A modern, AI-assisted merchant onboarding platform for Payment Aggregators with intelligent document processing, auto-fill capabilities, and real-time guidance.

## Features

- **Modern Landing Page**: Professional, responsive landing page with feature highlights
- **AI-Powered Assistance**: Real-time AI chatbot to guide merchants through the onboarding process
- **Smart Document Processing**: OCR-based document upload with automatic data extraction
- **Auto-Fill Functionality**: Automatically fills 80% of form fields from uploaded documents
- **Real-Time Validation**: Intelligent form validation with helpful error messages
- **Data Enrichment**: Auto-fetch business details from GST, PAN, IFSC, and Pincode APIs
- **Progress Tracking**: Visual progress indicators throughout the journey
- **Mobile Responsive**: Fully responsive design that works on all devices

## Quick Start

### Installation

1. Install dependencies:
```bash
# Backend
cd backend && npm install

# Frontend
cd ../frontend && npm install
```

2. Run the application:
```bash
# Terminal 1 - Backend
cd backend && npm run dev

# Terminal 2 - Frontend
cd frontend && npm run dev
```

3. Open http://localhost:5173 in your browser
